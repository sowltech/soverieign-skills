# Instagram Content Prompts

## 3. Instagram Bio Converter

```text
Write 5 Instagram bio options for a creator selling a digital product.

Use these inputs:
- Name:
- Product name:
- Target audience:
- Result / transformation:
- Main pain point:
- Brand tone:
- CTA destination (freebie / waitlist / shop / audit / DM / link in bio):
- Any credibility markers I can truthfully claim:

Rules:
- Each bio must be under 150 characters
- Each bio must sound human, clear, and warm
- No corporate language
- No pushy sales wording
- No fake authority
- Include who I help, what result they get, and a CTA

For each option, provide:
1. The bio
2. Character count
3. Why it works
4. Best fit for this style:
   - simple
   - authority
   - warm
   - creator
   - premium

Then give:
- Best overall option
- Best option for clicks
- Best option for trust
- Best option for conversions
```

## 4. Carousel Content Machine

```text
Create a 9-slide Instagram carousel that sells my digital product without sounding like an ad.

Use these inputs:
- Product name:
- Audience:
- Specific result it helps them achieve:
- One problem it solves:
- One myth or bad assumption in this space:
- One proof point or real example I can truthfully use:
- Comment keyword:
- Tone:
- CTA goal (comment / DM / click link in bio):

Rules:
- Do not invent news, quotes, or proof
- If no current event or quote is supplied, use a timeless hook
- Tone must feel warm, real, conversational, and high-trust
- Slides 2-6 must teach real value
- Slide 9 must use a soft CTA

Output:
- Carousel strategy angle
- Hook options (3)
- Slide-by-slide copy for all 9 slides
- Suggested text hierarchy per slide (headline / support line / CTA)
- Design direction for each slide
- Best caption to pair with the carousel
- Comment-to-DM bridge line
- Why this carousel converts

Structure:
- Slide 1: pattern interrupt / hook
- Slides 2-6: value teaching around one clear problem
- Slide 7: myth bust
- Slide 8: proof / example / transformation snapshot
- Slide 9: soft CTA to comment [keyword]

Keep it practical, saveable, and credible.
```

## 5. Sales Caption Writer

```text
Write an Instagram caption that sells my digital product without sounding salesy.

Use these inputs:
- Product name:
- Product type:
- Target audience:
- Main problem:
- Desired result:
- Why I created it:
- 3 things the buyer will be able to do after purchasing:
- One proof point I can truthfully mention:
- Comment keyword:
- Tone:
- Max word count:

Rules:
- Start with a one-line hook that stops the scroll
- Then tell a short personal story about why I created the product
- Then list 3 clear buyer outcomes
- Then include one line of truthful social proof or credibility
- End with a soft CTA to comment [keyword]
- Keep it human, grounded, and warm
- No buzzwords
- No fake hype
- No false urgency
- No invented proof

Output:
1. 3 hook options
2. Full caption
3. A tighter version under 150 words
4. A stronger conversion version that is still soft
5. Why the caption works

If proof is missing, label that section:
Proof placeholder to replace with real evidence
```

## 6. Story Selling Reel Script

```text
Write a 60-second Instagram Reel script that sells my digital product through storytelling.

Use these inputs:
- Product name:
- Audience:
- Everyday problem they face:
- My turning point moment:
- The solution I discovered:
- One real result or transformation:
- Comment keyword:
- Tone:
- On-camera style (talking head / voiceover / mixed):

Rules:
- No hard selling
- No fake hype
- Write the way a real person talks
- Make each section short enough for natural speech
- Include text-on-screen suggestions
- Include B-roll or visual suggestions if useful
- End with a soft CTA to comment [keyword]

Output format:
- Hook
- 0-10 sec
- 10-20 sec
- 20-35 sec
- 35-50 sec
- 50-60 sec
- Text on screen
- Suggested visuals
- Caption CTA
- Why this script converts

Structure:
1. Open with a relatable daily problem
2. Introduce me and the turning point
3. Reveal the solution
4. Share one specific real result
5. End with a CTA to comment [keyword]

Then provide:
- Version A: warmer
- Version B: more direct
- Version C: more premium / expert-led
```
