# DM Flows

## 7. DM Automation Sequence

```text
Write a 3-message Instagram DM sequence for someone who commented [keyword] on my post.

Use these inputs:
- Keyword:
- Topic of the original post:
- Product name:
- Audience:
- Result the product helps them achieve:
- Freebie / training / link being delivered:
- One real proof point or user outcome:
- Soft urgency reason, if truthful:
- Tone:

Rules:
- The messages must feel personal, warm, and helpful
- No pressure
- No aggressive selling
- No manipulative urgency
- No fake scarcity
- No invented proof
- Keep each message concise and natural
- Make Message 1 useful even if they never buy

Output:
### Message 1 - immediate reply
- Deliver the freebie or resource
- Make them feel seen
- Set up the next step naturally

### Message 2 - 24 hours later
- Share a short transformation story, insight, or common mistake
- Mention the product softly as the structured solution

### Message 3 - 48 hours later
- Invite them to buy with a soft urgency line only if truthful
- Make it easy to ignore without pressure

Then add:
- Branch reply if they say “interested”
- Branch reply if they say “too expensive”
- Branch reply if they do not respond
- Best CTA line
- Why this sequence works

Keep it sounding like a real human, not a marketing bot.
```

Add a lightweight DM logic map that shows:

## Quick Use Note
This note is for comment-to-DM conversion flows where trust matters more than pressure. A rare concept online, apparently.
